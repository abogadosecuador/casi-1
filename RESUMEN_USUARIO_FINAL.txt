================================================================================
                        CONFIGURACIÓN REQUERIDA
================================================================================

1. SUPABASE
   ├─ Crear proyecto en supabase.com
   ├─ Copiar SUPABASE_URL
   ├─ Copiar SUPABASE_ANON_KEY
   ├─ Ejecutar SUPABASE_SETUP.sql
   └─ Habilitar autenticación Email/Password

2. CLOUDFLARE
   ├─ Crear cuenta en cloudflare.com
   ├─ Crear namespace KV
   ├─ Crear base de datos D1
   ├─ Configurar wrangler.toml
   └─ Desplegar workers

3. VARIABLES DE ENTORNO (.env.local)
   ├─ VITE_SUPABASE_URL
   ├─ VITE_SUPABASE_ANON_KEY
   ├─ VITE_CLOUDFLARE_ACCOUNT_ID
   ├─ VITE_CLOUDFLARE_API_TOKEN
   ├─ VITE_BINANCE_API_KEY
   ├─ VITE_PAYPAL_CLIENT_ID
   └─ VITE_GEMINI_API_KEY

4. DEPENDENCIAS
   ├─ npm install
   ├─ Todas las dependencias están en package.json
   └─ Supabase, Framer Motion, TailwindCSS, etc.

================================================================================
                        CÓMO INICIAR
================================================================================

1. CLONAR REPOSITORIO
   git clone <repository-url>
   cd casi-1

2. INSTALAR DEPENDENCIAS
   npm install

3. CONFIGURAR VARIABLES DE ENTORNO
   cp .env.example .env.local
   # Editar .env.local con tus credenciales

4. CONFIGURAR SUPABASE
   # Ejecutar SUPABASE_SETUP.sql en Supabase SQL Editor

5. INICIAR SERVIDOR
   npm run dev

6. ACCEDER A LA PLATAFORMA
   http://localhost:3000/

7. REGISTRARSE
   ├─ Haz clic en "Registrarse"
   ├─ Completa el formulario
   └─ Comienza a usar la plataforma
